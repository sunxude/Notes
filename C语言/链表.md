链表是一种常见的[[数据结构]]，用于存储线性数据集合。在C语言中，链表由一系列节点组成，每个节点包含数据和指向下一个节点的指针。
#### 链表基础
```C
typedef struct Node
{
	int data;
	struct Node *next;
}Node;
```
定义了一个名为Node的结构体。
#### 单向链表
```C
Node *head = NULL;
Node *new_node = (Node *)malloc(sizeof(Node));
new_node->data = 42;
new_node->next = head;
head = new_node;
```
